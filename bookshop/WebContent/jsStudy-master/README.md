## 项目下载

	git clone git@github.com:airen/jsStudy.git

## 项目运行

先在本地执行:

	npm install

如果执行会遇到麻烦，可以尝试：

	sudo npm install

或者

	tnpm install

安装完依赖环境之后，可以直接运行：

	gulp

运行之后，浏览器会直接开启:

	http://localhost:3000/

项目首页存放在`app`目录下，在上面的地址后添加`app`

	http://localhost:3000/app

即可看到对应练习的地址和效果。
